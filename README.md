# 📖 Star Clothing App <a name="about-project"></a>

**Star Clothing App** is a e-commerce app that allows users to buy clothes online. It is built with React, Redux, Firebase, and Stripe.

## 🛠 Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>

- React.js
- Redux
- JavaScript
- CSS
- Jest
- Linters
- Firebase
- Stripe

<!-- Features -->

### Key Features <a name="key-features"></a>

- **Login/Register**
- **Add items to cart**
- **Buy items**
